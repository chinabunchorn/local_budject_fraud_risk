# ตัวอย่าง การคำนวณระยะทางและค่าขนส่งวัสดุก่อสร้าง

กรณีของงานก่อสร้างทางฯ ขนส่งวัสดุก่อสร้างด้วยรถบรรทุก 10 ล้อ ระยะทางขนส่งวัสดุจากแหล่งถึงกึ่งกลางหน้างาน 50 กม. เป็นทางราบลาดยาง 35 กม. ทางลาดยางลูกเนิน 3 กม. ทางลาดยางภูเขา 2 กม. และทางลูกรังราบ 8 กม. ทางลูกรังลูกเนิน 2 กม. ราคาน้ำมันดีเซล ลิตรละ 38 บาท

1.  ระยะทางขนส่งจากแหล่งวัสดุถึงกึ่งกลางหน้างาน = L = 50 กม.
2.  ทางลาดยาง
    *   ระยะทางราบ 35 กม. x 1.00 = 35.00 กม.
    *   ระยะทางลูกเนิน 3 กม. x 1.95 = 5.85 กม.
    *   ระยะทางภูเขา 2 กม. x 2.52 = 5.04 กม.
3.  ทางผิวลูกรัง
    *   ระยะทางราบ 8 กม. x 1.23 = 9.84 กม.
    *   ระยะทางลูกเนิน 2 กม. x 2.18 = 4.36 กม.
    *   ระยะทางภูเขา 0 กม. x 2.75 = 0 กม.
    *   รวมระยะทาง = D = 60.09 กม.
4.  ตัวแปรค่าขนส่ง = F = 60.09 / 50.00 = 1.2018
5.  ค่าขนส่ง
    *   ค่าขนส่งจากตารางค่าขนส่งที่ระยะทาง 50 กม. = 196.70 บาท/ลบ.ม.
    *   ดังนั้น ค่าขนส่ง สำหรับกรณีนี้ = 1.2018 x 196.70 = 236.39 บาท/ลบ.ม.
